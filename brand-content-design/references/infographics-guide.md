## Data Structure Reference

### Basic Data Format
```javascript
{
  title: "Main Title",           // Required: Headline
  desc: "Optional description",  // Optional: Subheadline
  items: [                       // Required: Array of items
    {
      label: "Item Label",       // Short identifier
      desc: "Description",       // Longer explanation
      value: 100,                // Optional: numeric value (charts)
      icon: "icon-name",         // Optional: icon reference
      children: [...]            // Optional: nested items (hierarchy/compare)
    }
  ]
}
```

### Comparison Data Format
```javascript
{
  title: "Before vs After",
  items: [
    {
      label: "Before",
      children: [
        { label: "Slow" },
        { label: "Manual" }
      ]
    },
    {
      label: "After",
      children: [
        { label: "Fast" },
        { label: "Automated" }
      ]
    }
  ]
}
```

### Chart Data Format
```javascript
{
  title: "Quarterly Revenue",
  items: [
    { label: "Q1", value: 120 },
    { label: "Q2", value: 150 },
    { label: "Q3", value: 180 }
  ]
}
```

## Theme Configuration

### Basic Theme
```javascript
themeConfig: {
  colorPrimary: "#3B82F6",     // Main brand color
  colorBg: "#FFFFFF",          // Background color
  palette: ["#3B82F6", "#10B981", "#F59E0B"]  // Color palette for items
}
```

### Stylize Options
```javascript
// Hand-drawn sketch style
themeConfig: {
  stylize: {
    type: "rough",
    roughness: 1.5,      // 0.5-3, higher = rougher
    bowing: 1,           // curve bowing
    fillWeight: 2,       // line thickness
    hachureGap: 4        // fill line gap
  }
}

// Linear gradient
themeConfig: {
  stylize: {
    type: "linear-gradient",
    angle: 90,           // 0=right, 90=down, 180=left, 270=up
    colors: ["#3B82F6", "#10B981"]
  }
}

// Radial gradient
themeConfig: {
  stylize: {
    type: "radial-gradient",
    colors: ["#3B82F6", "#10B981"]
  }
}

// Pattern fill
themeConfig: {
  stylize: {
    type: "pattern",
    pattern: "diagonal-stripe",
    scale: 1,
    foregroundColor: "#3B82F6",
    backgroundColor: "#FFFFFF"
  }
}
```

## Icons

The infographic generator includes 1900+ Lucide icons for use in templates that support icons.

### Using Icons

Add icons to data items using the `icon:` prefix:
```javascript
{
  items: [
    { label: "Research", desc: "Gather data", icon: "icon:search" },
    { label: "Design", desc: "Create mockups", icon: "icon:palette" },
    { label: "Build", desc: "Write code", icon: "icon:code" },
    { label: "Deploy", desc: "Go live", icon: "icon:rocket" }
  ]
}
```

### Templates with Icon Support

These templates render icons within items:
- `list-row-horizontal-icon-line` - Horizontal process with icons
- `list-row-horizontal-icon-arrow` - Horizontal flow with icons
- `list-column-vertical-icon-arrow` - Vertical flow with icons
- `list-grid-horizontal-icon-arrow` - Grid with icons
- `sequence-color-snake-steps-horizontal-icon-line` - Snake path with icons

### Searching Icons

Use the icon helper to search all 1900+ icons:
```javascript

// Search by keyword
searchIcons('chart');  // ['chart-bar', 'chart-line', 'chart-pie', ...]

// List categories
listCategories();  // ['process', 'business', 'growth', ...]
```
