# Acknowledgements

The infographic generator skill relies on the following open source libraries:

## Core Libraries

### @antv/infographic
- **License**: MIT
- **Author**: [Aarebecca](https://github.com/AntV)
- **Repository**: https://github.com/antvis/infographic
- **Description**: Infographic visualization library with 114+ templates across 7 categories (sequence, list, hierarchy, compare, quadrant, relation, chart)

### Lucide Icons (lucide-static)
- **License**: ISC
- **Website**: https://lucide.dev
- **Repository**: https://github.com/lucide-icons/lucide
- **Description**: 1900+ beautiful & consistent icons. Used for icon-based infographic templates.

## Rendering & Export

### jsdom
- **License**: MIT
- **Repository**: https://github.com/jsdom/jsdom
- **Description**: JavaScript implementation of web standards for Node.js. Provides DOM environment for server-side SVG rendering.

### Puppeteer
- **License**: Apache-2.0
- **Author**: The Chromium Authors
- **Repository**: https://github.com/puppeteer/puppeteer
- **Description**: Headless Chrome Node.js API. Used for high-quality PNG export from SVG.

### Sharp
- **License**: Apache-2.0
- **Author**: Lovell Fuller
- **Website**: https://sharp.pixelplumbing.com
- **Repository**: https://github.com/lovell/sharp
- **Description**: High-performance image processing. Used for image manipulation and format conversion.

### @resvg/resvg-js
- **License**: MPL-2.0
- **Repository**: https://github.com/nicholasxuu/resvg-js
- **Description**: High-performance SVG renderer based on resvg. Alternative rendering backend.

## Visualization Foundation

### D3.js
- **License**: ISC
- **Website**: https://d3js.org
- **Author**: Mike Bostock
- **Description**: Data visualization library (transitive dependency via @antv/infographic)

### Rough.js
- **License**: MIT
- **Website**: https://roughjs.com
- **Author**: Preet Shihn
- **Description**: Creates hand-drawn, sketchy graphics (transitive dependency for stylized templates)

---

We are grateful to all the maintainers and contributors of these projects for making this skill possible.
