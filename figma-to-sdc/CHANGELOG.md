# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-01-07

### Added

- Initial release
- `converting-figma-to-sdc` skill for AI-assisted conversion
- `/figma-to-sdc` command for explicit invocation
- Support for SDC (Twig) output
- Support for Canvas (JSX) output
- Figma MCP integration for data extraction
- Reference guides:
  - Figma extraction patterns
  - SDC generation patterns
  - Canvas generation patterns
  - Bootstrap accommodation framework
  - Prop type mapping
